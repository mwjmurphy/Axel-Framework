storageA "storage" definition is used by the framework to define one or more data repositories such as a Database.
databaseA "database" describes the tables, customised sql, functions and other features of a database that are used by the framework to map data between web pages and the database.
insertAn "insert" definition that inserts one database into another. This process occurs when the storage definition is being loaded at startup.
db_specificA "db_specific" definition is used for a specific database such as "oracle" or "mysql"
pk_createHow primary keys should be created.
sql"sql" is generic hand made sql that is available to the framework when using list's or queries.
functionUsed to build functions into the sql when retrieving data from a database
tableThe definition of a database table.
pkA primary key definition.
fkA foreign key definition.
textA definition for a text field. This is displayed as a single line. @See textarea for multiple line display.
textareaA definition for a text area field. This is displayed on multiple lines. @See text for a single line display.
passwordA definition for a password field.
linkA definition for a link (href).
imageA definition for a image (img).
intA definition for an integer field.
binaryA definition for a checkbox field. This works with a database binary field type. And can be display as a checkbox.
timestampA definition for a timestamp field. This is displayed as a single line.
datetimeA definition for a datetime field. This is displayed as a single line.
dateA definition for a date field. This is displayed as a single line. The value of datetime is YYYY-MM-DD
timeofdayA definition for a timeofday field. This is displayed as a single line. The value of datetime is HH:MM:SS or HH:MM
selectMulti Option Selection used by a radio button display with single or multi select option.
Use for entries such as "Y,N" or "Y,N,U" or "Yes,No,Maybe,Perhaps" or "true,false" or "Fast Car;Slow Car;"
table_pathA table path is used to identify relationship tables when a "fk" (foreign key) is not used directly between the two tables and a link table is used instead.
