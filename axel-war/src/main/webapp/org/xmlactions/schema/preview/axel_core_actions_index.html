codeThe code action will invoke Java code from the web page.
httpThe http action will make a http rest call to the url.
paramProvides parameters for some of the actions such as code, http or transform
insertInsert files into a page.
insert_intoInsert the current processing page into another page.
echoEchos content to page
ifA conditional expression evaluation in javascript.
elseifA conditional elseif expression evaluation in javascript.
elseConditional elseif expression evaluation if true executes enclosing actions. The else action is the latter part of the if, elseif, else actions.
putPut a variable into a map. Can be retrieved with ${key}. The element content is the value for the variable.
removeRemove a variable from the execContext
getGet a variable from the execContext map.
json_getGet an object/s from a json document.
evalEvaluate a javascript expression.
date_formatterConvert a date from one format to another format.
transformTransform an xml document using an xsd style sheet.
escapeThis action will escape data to a number of different formats.
unescapeThis action will unescape data from a number of different formats.
debugDisplay debug information on the presentation page.
logLogs the content of the ExecContext to output.
rangeCreates a Range bean that provides an incremental/decremental index or a random index.
map_xml_to_beanMaps an xml to a java bean.
map_bean_to_xmlMaps a java bean to xml.
scriptThe "script" action will execute a javascript.
map_xml_to_presentationMap selected content of an XML to a presentation form.
map_json_to_presentationMap selected content of a JSON object to a presentation form.
map_xml_to_jsonMap XML to JSON
content_typeThis actions allows the page to change the http request content type.
