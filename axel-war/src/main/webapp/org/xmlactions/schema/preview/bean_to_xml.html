bean - element

index

A bean defines the properties of a class included in the xml mapping.

attributes

id - attribute

type - xsd:string
use - required

The Bean Identifier. This is used when a bean references another bean.

clas - attribute

type - xsd:string
use - required

The name of the Java Class that this bean represents.

name - attribute

type - xsd:string
use - optional

The xml element that will be created from this bean.

This is an optional attribute, if not used then the id attribute will be used instead.

prefix - attribute

type - xsd:string
use - optional

Use as a namespace prefix, prepended to the name when creating the element name.

Note that an assumption is not made to place the ':' character between the prefix and name. This must be included with the prefix if required.

attribute - element

index

Definition of a bean xml attribute

attributes

property - attribute

type - xsd:string
use - required

This is the name of the bean property that this attribute references.

If the name property is not set this value will be used instead as the attribute name.

name - attribute

type - xsd:string
use - optional

The name of the xml attribute that will be created.

This is an optional attribute, if not used then the property value is used instead.

populator - attribute

type - xsd:string
use - optional

A String representation of a Java class that is used to populate the xml from the bean property.

Used when a property to xml needs to be formatted or handled as a list, timestamp, date etc.

prefix - attribute

type - xsd:string
use - optional

Use as a namespace prefix, prepended to the name when creating the attribute name.

Note that an assumption is not made to place the ':' character between the prefix and name. This must be included with the prefix if required.

populator_ref - attribute

type - xsd:string
use - optional

A reference to a populator.

Used when a property to xml needs to be formatted or handled as a list, timestamp, date etc.

populator - element

index

A populator is used to set the derived bean property in the xml. Used when the object to be set in the xml is stored into a list, a date, a timestamp etc.

A set of populators are delivered with the product, but additional ones may be created and referenced from the configuration once they are set in the java classpath.

attributes

id - attribute

type - xsd:string
use - required

The reference identifier for this populater. Referenced from the property populator_ref attribute.

clas - attribute

type - xsd:string
use - required

The name of the Java Class that performs the population.

child elements

ref- keyvalue
minOccurs - 0



keyvalue - element

index

A keyvalue is used to store parameters for populators.

Example. when setting a timestamp or a date the format of the date may be set using key="time_format" value="yyyy-MM-dd HH:mm:ss.SSS"

attributes

key - attribute

type - xsd:string
use - required

The key to identifying this keyvalue

value - attribute

type - xsd:string
use - required

The value for the key.
