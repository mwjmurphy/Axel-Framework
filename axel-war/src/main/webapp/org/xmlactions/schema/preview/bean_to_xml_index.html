bean_to_xmlThe root element for all bean to xml mapping.
beanA bean defines the properties of a class included in the xml mapping.
attributeDefinition of a bean xml attribute
elementDefinition of a bean child element.
populatorA populator is used to set the derived bean property in the xml. Used when the object to be set in the xml is stored into a list, a date, a timestamp etc.
keyvalueA keyvalue is used to store parameters for populators.
namespaceWhen we want to add namespace identifiers to the output xml.
