  1. If and Else

Conditional Statement Actions


Examples showing axel:if & axel:elseif & axel:else in use

The if, elseif and else statements are used to make decisions on the page as it is being constructed.

The order for building the conditional statements is

  • if - required and can be only one
  • elseif - optional and there can be many
  • else - optional and can be only one

INFO: The nesting syntax of the actions is that the axel:if is always the parent and the axel:elseif and axel:else are always it's children. But for viewing the conditional statement actions it's easier to place them all on the same indent as is shown in the examples below.

<axel:if expression="1 == 1"> 1 is the same as 1 </axel:if>
An axel:if example
${if_action}
The output is

    1 is the same as 1
<axel:if expression="1 == 2"> 1 is the same as 1 <axel:else> 1 is not the same as 2 </axel:else> </axel:if>

<axel:if expression="1 == 2"> 1 is the same as 2 <axel:elseif expression="2 == 2"> 2 is the same as 2 </axel:elseif> <axel:else> Not if or elseif </axel:else> </axel:if>
