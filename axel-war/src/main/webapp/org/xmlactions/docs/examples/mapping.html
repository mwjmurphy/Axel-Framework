To map an XML file to java beans you need to create a definition file that provides the instructions for mapping the XML to java. Documentation on mapping XML to Beans can be seen on the main page Mapping xml to java

Mapping Beans to XML


To map Java Beans to XML you need to create a definition file that provides the instructions for mapping the Java Beans to XML. Documentation on mapping Beans to XML can be seen on the main page Mapping beans to java
