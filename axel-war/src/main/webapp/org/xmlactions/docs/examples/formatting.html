  1. Formatting Data

Formatting Dates


Examples showing axel:date_formatter in use

The date_formatter is used to map the date format to a different date format.


An axel:date_formatter example
<axel:date_formatter
    date_value="1995-01-25"
    origin_format="yyyy/MM/dd"
    destination_format="dd/MM/yyyy" />
The output

Escaping Data


Examples showing axel:escape in use

The escape is used to escape data to a number of different formats. See "apache commons escape" for an explanation on how this works.


An axel:escape example
<axel:escape
    ref_key="escape_this_data"
    format="html"/>
