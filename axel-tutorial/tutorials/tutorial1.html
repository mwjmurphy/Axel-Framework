This first tutorial shows how to download the axel-run application which is used for all the tutorials.

Show "Hello World!!! time" in your browser.
${instruction-counter}. Download the latest version of axel-run jar file from this link ${axel_run_version}.jar
${instruction-counter}. From the command line type java -jar ${axel_run_version}.jar
${instruction-counter}. When the application has started open your web browser and enter localhost - you'll see an error "Whitelabel Error Page"
${instruction-counter}. In the same location, create a new file called index.html and edit the file adding
                
                    
Hello World!!! ${current.time}
${instruction-counter}. In your web browser now type localhost/index.html and you'll see. The time show here will have changed to show your time.
                
Hello World!!! ${current.time}
You've just created your first html page with axel that shows Hello World!!! with your local time.
