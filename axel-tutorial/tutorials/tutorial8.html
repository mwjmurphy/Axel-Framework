8 example - json get

Get an object/s from a json document.

Get an object/s from a json document that already exists in the execContext.

  1. Edit the index.html page and add the following content 

    <axel:json_get json_data="${people_json}" json_path="name"/>

  2. Refresh the browser and now you see the new content from the json document.

name:

  1. Edit the index.html page and add the following content, note the index="1" 

    <axel:json_get json_data="${people_json}" json_path="name" index="1"/>

  2. Refresh the browser and now you see the new content from the json document showing the second name.
