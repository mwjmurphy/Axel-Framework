10 example - put and get

How to put, get and remove content from execContext

Put some content into execContext and Get some content from execContext and Remove some content from execContext

  1. Edit the index.html page and add the following content 

    <axel:put key="a_key">This is the value stored for a_key</axel:put>
<axel:get key="a_key"/><
<axel:remove key="a_key"/>
<axel:get key="a_key"/>

    2. Stores a value for "a_key" into the execContext using axel:put then uses axel:get to retrieve it, then uses axel:remove to remove it.

  2. Refresh the browser and now you see the new content from the json document.
