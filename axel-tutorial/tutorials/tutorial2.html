2 example - insert component page

Inserting a component page into the index.html

  1. Create a new file called header.html and add the following content 

    <h5>This is an Axel Framework Header Component Page</h5>

  2. Edit the index.html page and add the following content 

    <axel:insert page="header.html"/>

  3. Refresh the browser and now you see

Hello World!!! ${current.time}
This is an Axel Framework Header Component Page
