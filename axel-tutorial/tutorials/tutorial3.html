3 example - capture browser parameters

How to access the parameters from the browser

  1. Edit the index.html page and add the following content 

    <h5>Hello ${request:title} ${request:surname}</h5>

  2. In your browser type localhost/index.html?title=Mr&surname=Flinstoneb

  3. Now you see this in your browser

Hello World!!! ${current.time} This is an Axel Framework Header Component Page Hello Mr Flinstone
