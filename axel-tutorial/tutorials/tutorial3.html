${tutorial_page_index} example - capture browser parameters

How to access the parameters from the browser using axel

Accessing parameters from a browser request.
${instruction-counter}. Edit the index.html page and add the following content
                
                    

Hello ${request:title} ${request:surname}
Shows Hello with two request parameters named "title" and "surname"
${instruction-counter}. In your browser type localhost/index.html?title=Mr&surname=Flinstone
                

Hello Mr Flinstone
Now you can the two paraameters on screen instead of the replacement markers ${request:title} ${request:surname}
